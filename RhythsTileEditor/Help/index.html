Rhyth's Tile Editor help page

Rhyth's Tile Editor is an editor for the RhyTiles library. It makes it easy to design and save tilemaps and later import them into a game engine of your choice. Simply integrate this DLL and you are ready to load a tileset and map.

Tilesets

Tilesets are the building blocks you use to create maps. You first have to draw or find an image you would like to use. The tileset cuts the image into defined pieces. The tiles in the image must have the same size and have no offset or padding.


To interact with the tileset, you have to use the right bottom part of the editor. Here you can select a tile from the current tileset and create or load tilesets.


The buttons in order from left to right do the following:


New - This opens a dialog to help you create a new tileset.

Load - This opens a window where you can select a previously saved tileset.

Save - Save the current tileset to a previously saved location.

Save as - Save the current tileset to a newly specified location.

Close - Close the currently selected tileset.

Creating a new tileset


When creating a new tileset, you have to set all properties needed in the dialog that opened up.


Here is an explanation for each property:


Name - This makes it easier to identify which tileset is currently loaded. A name does not have to be unique to a tileset, but it is recommended that each name is different.

Source - The source is the reference to the image that should be used. You can either type in a path directly or press the button next to the source to open a dialog to select a specific file.

Dimensions - The width and height represent the width and height in pixels that a tile in a tileset has.

Tileset Viewer


The little window that is displaying your tileset is controlled with the mouse. To pan around the tileset, hold the middle mouse button and drag. To select a tile, press the left mouse button. To unselect a tile, either select press outside the tileset or press the right mouse button.


When you left-click a tile, it becomes highlighted. A highlighted tile is shown by having a blue tint on it. The highlighted tile is used when you use a tool in the Map Viewer window.


If you have more than one tileset opened, then you can use the tabs above the screen to change between them.

Maps

The main window displays the map that you are working on.


Here, you can place tiles from tilesets and slowly build up your map.


The buttons on the top do the following:


New - This opens a dialog to help you create a new map.

Load - This opens a window where you can select a previously saved map.

Save - Save the current map to a previously saved location.

Save as - Save the current map to a newly specified location.

Close - Close the currently selected map.


Brush - Paint a single tile onto the map.

Eraser - Delete a single tile from the map.

Flood Fill - Replace all matching tiles from the map.


Undo - Undo the last action.

Redo - Redo the last undo.

Creating a new map


When creating a new map, you have to set all properties needed in the dialog that opened up.


Here is an explanation for each property:


Name - This makes it easier to identify which map is currently being edited. A name does not have to be unique to a map, but it is recommended that each name is different.


Tile Width/ Tile Height - The tile width and tile height represent the width and height in pixels that a tile on a map has. If a tile that has different dimensions is put onto the map then the tile gets streteched to fit into the tile.

Width/ Height - The width and tile height represent the amount of tiles are on the map.

Map Viewer


The map viewer has similar controls to the tileset viewer. To pan around the map, hold the middle mouse button and drag. To use your current select tool, press the left mouse button. The tool will be applied to the clicked tile on the map.


If you have more than one map opened, then you can use the tabs above the screen to change between them.

Layers


Layers are virtually maps put on top of other maps. The first layer is drawn first. This means that you can have transparent tiles on the above layers to make a detailed looking map.


The layers can be found on the right-hand side above the tileset viewer.


When you are editing a map, you always draw on the currently selected layer. To select a layer, press on one of the layers listed.


Here is an explanation for each button. You can also right click on a layer to bring up a menu with all the buttons inside it or double click a layer to quickly rename it.


New - This creates a new layer.

Remove - This removes the currently selected layer.

Rename - Renames the current selected layer.

Toggle visibility - Toggle visibility on the currently selected layer. Only visible layers are drawn on the map.

Move layer up - Moves the currently selected layer up by one.

Move layer down - Moves the currently selected layer down by one.

Config


The config menu is brought up by pressing the "Config" button on the tool strip.


In here, you can adjust settings that are saved automatically when you press the "close" button.


If at any point you are unhappy with the current configuration you can press the "Reset" button to reset the configuration to the default one.

Shortcuts


Here are keyboard shortcuts to make working in the editor easier:


Undo (CTRL + Z) - Undo the last action.

Redo (CTRL + Y) - Redo the last undo.

Save (CTRL + S) - Save the current map to a previously saved location.

Save as (CTRL + SHIFT + S) - Save the current map to a newly specified location.

Close (CTRL + W) - Close the currently selected map.

Load - This opens a window where you can load a previously saved map.


Brush (B) - Select the brush tool.

Eraser (D) - Select the eraser tool.

Flood Fill (F) - Select the flood fill tool.


These shortcuts are only for dialogs:

Enter (Accept) - Quickly accept the dialog.

Close (Cancel) - Quickly cancel the dialog.

Quickly open a file


You can quickly open a map or tileset by simply dragging it from the windows explorer right into the application window. No more browsing through menus!
